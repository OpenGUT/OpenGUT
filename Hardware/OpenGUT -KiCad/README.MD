# OpenGUT – KiCad PCB Design

This repository contains KiCad design files for the **OpenGUT** hardware platform.  
It extends the original OpenGUT repository by providing open-source PCB layouts and schematics created in KiCad.

---

## ✨ Contribution Highlights
- **KiCad PCB Design Files**: Complete board design using KiCad, ensuring accessibility and reproducibility in an open-source EDA toolchain.
- **IMU Integration**: Added an Inertial Measurement Unit (IMU) to support **noise cancellation** and enhanced sensing capabilities.
- **Open-Source Alignment**: Since OpenGUT is an open-source platform, developing its PCB in KiCad strengthens transparency, collaboration, and community-driven improvements.
